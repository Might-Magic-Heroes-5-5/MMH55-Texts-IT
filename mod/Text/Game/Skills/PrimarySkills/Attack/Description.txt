L'Attacco dell'Eroe aumenta il danno inflitto dagli attacchi dell'esercito dell'eroe aumentando il loro Attacco. Migliora anche l'efficacia della Balista.<br><br><color_default>Calcolo del danno: se A è più grande di D, danno inflitto = AssaultingStack_Size * Random(AssaultingStack_MinDamage, AssaultingStack_MaxDamage) * [1 + 0.025*(A-D)]; se D è maggiore di A, danno inflitto = AssaultingStack_Size * Random(AssaultingStack_MinDamage, AssaultingStack_MaxDamage) / [1 + 0,025*(D-A)].
