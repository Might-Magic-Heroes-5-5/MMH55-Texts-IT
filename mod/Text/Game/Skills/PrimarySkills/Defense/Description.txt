La Difesa dell'eroe diminuisce il danno subito dall'esercito dell'eroe aumentando la loro Difesa.<br><br><color_default>Calcolo del danno: se A è più grande di D, danno inflitto = AssaultingStack_Size * Casuale(AssaultingStack_MinDamage, AssaultingStack_MaxDamage) * [1 + 0,025*(A-D)]; se D è maggiore di A, danno inflitto = AssaultingStack_Size * Random(AssaultingStack_MinDamage, AssaultingStack_MaxDamage) / [1 + 0,025*(D-A)].
